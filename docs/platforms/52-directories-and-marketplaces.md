## 7. Rules

| Rule | Detail |
|------|--------|
| Free tiers only | Do not pay for premium listings, verified badges, or sponsored placements until business has revenue |
| Consistent branding | Use the same logo, tagline, and company description across all platforms |
| Real content only | No fake reviews, manufactured testimonials, or inflated project counts |
| Respond within 24 hours | All inbound leads from directories should get a response within 24 hours |
| Qualify before investing time | Minimum EUR 3K project value for custom dev leads |
| Update app count | All profiles should reflect "9 production apps" (check and update as new apps ship) |
